# Vocab Trainer 3 - Pierwsze Wydanie (v0.1.0)

## 🎉 Pierwsze wydanie aplikacji Vocab Trainer 3!

Vocab Trainer 3 to nowoczesna aplikacja do nauki słówek angielskich z polskim tłumaczeniem, zbudowana przy użyciu Tauri + React + TypeScript.

## ✨ Główne funkcje

### 📚 Zarządzanie słownictwem
- **Import słów**: Łatwy import słówek z plików JSON
- **Organizacja**: Przechowywanie słów z tłumaczeniami, wymową i przykładami zdań
- **Kategorie**: Możliwość organizowania słów w grupy

### 🧠 Inteligentne powtarzanie (SRS)
- **Algorytm SM-2**: Naukowo sprawdzony system spaced repetition
- **Poziomy trudności**: Again, Hard, Good, Easy
- **Statystyki**: Śledzenie postępów nauki

### 🔔 Inteligentne powiadomienia
- **Automatyczne przypomnienia**: Powiadomienia w optymalnych odstępach czasu
- **Konfigurowalne harmonogramy**: Ustawienia godzin pracy i przerw
- **Responsywny design**: Okno powiadomień skaluje się do rozmiaru ekranu

### 🎨 Nowoczesny interfejs
- **Responsywny design**: Działa na różnych rozmiarach ekranów
- **Ciemny motyw**: Przyjazny dla oczu interfejs
- **Animacje**: Płynne przejścia i efekty wizualne

## 🔧 Technologie
- **Frontend**: React 19 + TypeScript + Vite
- **Backend**: Tauri (Rust) + SQLite
- **UI**: CSS z nowoczesnymi efektami
- **System**: Windows (kompilacja dla innych platform w przygotowaniu)

## 📦 Instalacja

1. Pobierz plik `vocab-trainer3_0.1.0_x64-setup.exe`
2. Uruchom instalator
3. Aplikacja zostanie zainstalowana i będzie dostępna w menu Start

## 🚀 Pierwsze kroki

1. **Uruchom aplikację** - pojawi się ikona w tray'u systemowym
2. **Dodaj słowa** - użyj przycisku "Importuj słowa" lub dodaj ręcznie
3. **Rozpocznij naukę** - aplikacja zacznie wysyłać inteligentne powiadomienia
4. **Konfiguruj ustawienia** - dostosuj harmonogram powiadomień w ustawieniach

## 📋 Wymagania systemowe

- **System operacyjny**: Windows 10/11
- **Pamięć RAM**: Minimum 512MB
- **Miejsce na dysku**: ~50MB
- **Połączenie internetowe**: Nie wymagane (aplikacja działa offline)
